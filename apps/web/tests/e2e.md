# 前端 E2E 测试用例（设计与步骤）

## 1. 访问与渲染
- 打开 http://localhost:8000/，页面标题显示“TodoList”，无横向滚动；
- 三列看板渲染：待办/进行中/已完成，卡片数量与示例数据一致；

## 2. 搜索与交互
- 输入框可输入“设计系统”，列表过滤出匹配卡片（原型演示）；
- 悬停/点击卡片有视觉反馈，cursor-pointer 指示；

## 3. PWA 与离线
- 检查 manifest 加载与 service worker 注册；
- 离线后刷新仍可加载基础资源（index.html、manifest.json）；

## 4. 可访问性
- 键盘 Tab 可逐步聚焦到“搜索输入”“添加任务”按钮；
- 焦点可见环；

## 5. 响应式断点
- 375/768/1024/1440 视口下布局正常，无遮挡与溢出；
