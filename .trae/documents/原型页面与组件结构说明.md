# 原型页面与组件结构说明

## 页面与帧（Frames）
- 认证：`pg/Login`（1440×1024/375×812）、`pg/Register`、`pg/Forgot`
- 主视图：`pg/Todos-Desktop`（1440×1024）、`pg/Todos-Mobile`（375×812）
- 任务编辑：`pg/Todo-Edit`（Desktop Modal 640×720 / Mobile Bottom Sheet 高度自适应）
- 分类管理：`pg/Categories`（列表页 + 弹层）
- 访客模式：`pg/Visitor-Intro`（Banner说明 + 操作入口）
- 导出：`pg/Export-Dialog`（JSON/CSV，范围单选/复选）
- 设置：`pg/Settings`（主题、通知授权、同步状态）
- 系统：`pg/Empty`、`pg/Error`、`pg/Offline`

## 布局规范
- 桌面：主容器宽 1200，左右内边距 24，顶部 AppBar 高 64
- 左列（筛选区）宽 280，右列（列表区）自适应；卡片间距 16
- 移动：顶部导航高 56，内容区左右内边距 16，底部主要操作栏高 56

## 组件与变体（Variants）
- `cmp/AppBar`：默认/滚动阴影；含 `SearchBar`、`UserMenu`
- `cmp/Tabs`：active/inactive/disabled；指示条宽度随文本
- `cmp/SearchBar`：default/focus/loading/with-clear；占位“搜索待办…”
- `cmp/TaskCard`
  - 内容：`Checkbox`、`Title`、`Description`、`PriorityBadge`、`DueAt`、`Categories`
  - 状态：`default`、`hover`、`selected`、`done`（删除线+淡化） 、`expired`（错误色边框）
  - 交互：卡片点击进入编辑，`Checkbox` 切换完成
- `cmp/PriorityBadge`：`high|medium|low` 三态颜色
- `cmp/DatePicker`：日期/时间二级面板，过期禁用态
- `cmp/CategoryChip`：default/selected/closeable（带删除）
- `cmp/FilterChips`：状态/分类/时间范围；可多选
- `cmp/Modal`：支持标题、说明文本、主次按钮；移动端 `Bottom Sheet`
- `cmp/EmptyState`：插画占位/说明文字/去创建按钮
- `cmp/Toast`：成功/错误/信息；3秒自动消失
- `cmp/OfflineBanner`：离线提示/队列长度/立即同步按钮

## 命名与组织
- Pages：`Foundation`、`Components`、`Desktop`、`Mobile`、`Prototype`
- 节点命名：`pg/*`、`cmp/*`、`flow/*`，如 `flow/Add-Todo`
- 图层分组：`Header`、`Filters`、`List`、`Dialogs`

## 资源与Token引用
- 颜色/字体/间距/圆角/阴影引用自 `design/设计令牌.json`
- 将颜色与文本样式注册为 Figma Variables 与 Text Styles，间距作为 Auto Layout 参数

## 连接与原型设置
- 在 `Prototype` 页面创建关键流程的交互链接：点击、切换、状态变体
- 动画：Tabs 切换使用 `Smart Animate`，对话框使用 `Move in`（200ms）
- 设备预设：Desktop 1440×1024、Mobile iPhone 13（375×812）

## 验收与检查点
- 所有页面均有桌面与移动版本
- 组件变体覆盖状态完整，可复用性良好
- 原型可点击演示：快速添加、标记完成、分类筛选、导出、离线提示
